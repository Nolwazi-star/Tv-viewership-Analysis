# Tv-viewership-Analysis
Analysis of BrightTV user and session data to uncover viewing trends, factors influencing consumption, and provide recommendations to increase engagement and grow the subscriber base.

## Objectives
- Understand user and usage trends on BrightTV.
- Identify factors influencing content consumption.
- Recommend content strategies to boost engagement during low-traffic periods.
- Suggest initiatives to grow BrightTV’s subscriber base.

  ## Key Analysis Steps
1. **Data Cleaning & Preparation**  
   - Handle missing or inconsistent data
   - Convert timestamps from UTC to SAST
2. **Exploratory Data Analysis (EDA)**  
   - Identify trends in user activity and content consumption
3. **Insights & Recommendations**  
   - Analyze low-consumption days and propose content strategies
   - Suggest growth initiatives for increasing the subscriber base
  
     ## Tools & Technologies
- **Miro** – Project planning and workflow mapping
- **SQL** – Data extraction and querying
- **Excel** – Data visualization and analysis
- **Canva** – Final presentation design

  
## Deliverables
- 15-minute presentation summarizing insights and recommendations
- Visualizations showing user trends and content consumption patterns
- Actionable strategies for the CVM team
